## 两两交换链表中的节点
假设有x,y,z三个节点,它们之间的关系是这样的:  
x->y->z  
交换x与y的位置后,它们之间的关系是这样的:  
y->x->z

完成交换操作只需:`x->next = y->next;	y->next = x;`
这样就完成了两个节点之间的交换,接下来的重点是思考该如何在交换过后的各组节点之间进行连接.

假设有如下一组节点:
a->b->c->d->NULL
完成两两交换后节点之间的关系为:
- d->c->NULL  
  b->a->c->NULL

如果只是完成交换,显然不符合题目要求.   

使用递归解决问题:
- 终止条件为链表为空或者链表只剩一个元素  
- 返回值应为交换完成的一组节点,因此可以将返回值与前一组节点的尾端相连

递归结束后即完成所有交换
